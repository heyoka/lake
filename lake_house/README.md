# LakeHouse

An Elixir RabbitMQ Streams connector.

## Installation

```elixir
def deps do
  [
    {:lake_house, "~> 0.1.0"}
  ]
end
```

## Running Tests

With `LOCAL_LAKE=1`, the `lake` dependency is used directly from this
repository. Use this when developing something for `lake_house`.
